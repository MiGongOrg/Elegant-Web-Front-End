Elegant Web Front End

HTML / CSS / JavaScript

Elegant Web Front End documentation is built using GitBook.

Global installation GitBook

Please install nodejs and npm before continuing installation.

npm install -g gitbook-cli

Clone

git clone https://github.com/MiGongOrg/Elegant-Web-Front-End.git

Install

cd Elegant-Web-Front-End
gitbook init
gitbook install ./

Build

gitbook build

Preview

gitbook serve

http://localhost:4000
